Listas OL e UL 📝

Listas Ordenadas

Vamos criar uma lista ordenada usando a tag OL, <ol> junto da tag LI <li> para identificar cada item.

  1. Arroz
  2. Farinha
  3. Feijao

Além disso também podemos usar listar com formatos diferentes de ordem por exemplos

  1. Ordem numérica, ex: 1, 2, 3, 4, 5...
  2. Números romanos (Maiusculo), ex: I, II, III, IV, V...
  3. Número romanos (Minusculo), ex: i, ii, iii, iv, v...
  4. Letras (Maiusculo), ex: A, B, C, D, E...
  5. Letras (Minusculo), ex: a, b, c, d, e...

Fora isso também conseguimos começar por uma ordem diferente utilizando:
<ol type="I" star = "7">

    ficando assim:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

Listas Nao Ordenadas

Também podemos criar listas nao ordenadas usando a tag UL, <ul>

  • Utiliza
  • Bolinhas
  • para
  • Ordenar

Da mesma forma que o OL pode modificar os formatos da lita o UL também tem suas variaçoes.

    Desta forma

  • Bolinha pintada
  • Bolinha sem preechimento
  • Um pequeno quadrado
